<p>&nbsp;</p>
<p>java:</p>
<p>int[][] &nbsp;arr = new int[5][6];</p>
<p>java 可以同时指定两个维度的长度.</p>
<p>&nbsp;</p>
<p>c# 只能这样</p>
<p>int[][] &nbsp;arr = new int[5][];</p>
<p>c# 只能指定第一维的长度.</p>
<p>&nbsp;</p>
<p>但是java也支持像c#一样的写法, 只指定一维的长度:</p>
<p>int[][] &nbsp;arr = new int[5][];</p>
<p>&nbsp;</p>
<p>其实c#有自己的的二维数组数据结构:</p>
<p>int[,] arr = new int[5,6];</p>
<p>注意中间有逗号.</p>
<p>&nbsp;</p>
